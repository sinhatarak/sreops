# Featured SRE & Kubernetes Tips

1. **Kubernetes Pod Debugging**  
   Use `kubectl describe pod` + `kubectl logs` for quick troubleshooting.

2. **Alerting Best Practices**  
   Avoid alert fatigue: group alerts and tune thresholds.

3. **Disaster Recovery DR Drill**  
   Simulate failures safely using canary deployments or staging clusters.

4. **Monitoring Dashboards**  
   Use Grafana dashboards for quick insights into cluster health.
